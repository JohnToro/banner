glFusion Banner Plugin Configuration
glFusion Wiki
Banner ID
Each banner must have a unique ID. If one is not entered here, an ID will be automatically generated when the banner is saved.
Title
Each banner requires a title. This will be displayed in the administrative lists to help identify the banner.
Category
Each banner belongs to exactly one category. The category identifies the placement of the banner.
Campaign
Each banner belongs to exactly one advertising campaign.
Topic
Select which topic pages should show this banner, or All
Publish Start
If this banner should be published only after a certain date, check this checkbox and set the starting date and time. You can either select the date from the selections or use the datepicker by clicking on the calendar icon.

This value will be overridden by a later value associated with the campaign, if any.

Publish End
If this banners should stop being displayed after a certain date, check this checkbox and set the ending date and time. You can either select the date from the selections or use the datepicker by clicking on the calendar icon.

This value will be overridden by an earlier value associated with the campaign, if any.

Type of Ad
Select the type of ad for this banner. The options are:
  • Uploaded Image: An image is uploaded to this server and served from there.
  • Remotely-Hosted Image: The ad image will be served from a remote server. You simply specify the URL to the remote image.
  • HTML or Javascript: Enter the actual HTML or Javascript code needed to create the ad. This would be used for ad services such as Google Adsense.
Enabled?
If this is unchecked, then this banner will not be shown.
Hits
This field allows you to pre-populate or adjust the number of hits, or "click-throughs", that this banner has received.
Max. Hits
Enter the maximum number of hits that this banner may receive. After this number is reached, the banner will no longer be displayed. Enter Zero to allow unlimited hits.
Impressions
This field allows you to pre-populate or adjust the number of impressions for this banner. An "Impression" is recorded dach time the banner is displayed.
Max. Impressions
Enter the maximum number of times that this banner will be displayed. After this number is reached, the banner will no longer be shown. Enter Zero to allow unlimited impressions.
Target URL
Enter the URL where visitors will be redirected after clicking on an ad. This is required for images that are hosted locally or remotely, but not for HTML or Javascript-based ads since those will contain the url themselves.
Target
Select the target browser window for the redirection.
Upload a Banner
Select an image to upload for a locally-hosted banner. Does not apply to other banner types.
URL to a remote image
Enter the full url to a remotely-hosted banner image. Does not apply to other banner types.
Complete Ad Code
Enter the complete code needed for HTML or Javascript ads. Does not apply to other banner types.
Dimensions
Enter the width and height for the image (does not apply to HTML/Javascript banners). These values are not required, but are recommended, especially for remotely-hosted banners. For locally-hosted images, these values are calculated automatically if left blank.
Alt Tag
Enter the HTML "alt" tag for this banner. This is useful for visitors using screenreaders and other non-graphical browsers.
Weighting
This value indicates a relative priority for this banner. Banners with a higher value will be displayed more often, on average, than those with a lower value.
Owner
Administrators can set the owner ID for the banner. For non-admins this simply displays the owner's name
